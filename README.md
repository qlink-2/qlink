# qlink
A full rewrite of the original Qlink app that started it all.

## Build & run
Make sure you're on Flutter 2.0 before running `flutter run`.

## Project structure
```sh
├── android/
├── ios/
├── web/
├── lib/
│   ├── screens/
│   ├── theme.dart
│   └── main.dart
├── test/        *** "flutter test"
└── pubspec.yaml *** "flutter pub get"
```
